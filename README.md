# 🔐 Password Generator

## Project Overview

This is a desktop application built using Python and Tkinter that generates secure and customizable passwords. Users can choose the password length and include letters, numbers, and symbols based on their preferences. The application also provides a one-click option to copy the generated password to the clipboard.

---

## Features

- Generate random passwords
- Choose password length
- Include letters
- Include numbers
- Include symbols
- Copy generated password to the clipboard
- Input validation
- Clear input fields

- Tkinter
- Random

## Installation

1. Clone the repository.

```bash
git clone <repository-link>
```

2. Navigate to the project folder.

```bash
cd Password-Generator
```

3. Install the required packages.

```bash
pip install -r requirements.txt
```

4. Run the application.

```bash
python password_generator.py
```

---

## How to Use

1. Enter the desired password length.
2. Select the character types you want to include:
   - Letters
   - Numbers
   - Symbols
3. Click **Generate Password**.
4. Copy the generated password using the **Copy Password** button.
5. Click **Clear** to reset the application.
